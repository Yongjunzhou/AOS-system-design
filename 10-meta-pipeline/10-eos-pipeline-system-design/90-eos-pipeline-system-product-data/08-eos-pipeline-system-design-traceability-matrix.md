## 1:1 分配约束检查

### 检查结果

| 层间映射 | 检查项 | 结果 |
|---------|-------|------|
| OR末级 → SR架构末级 | 每条 OR 末级分配到唯一 SR 架构末级 | ✅ 通过 |
| SR详细末级 → SysReq架构末级 | 每条 SR-F 详细末级分配到唯一 SysReq-F 架构末级 | ✅ 通过 |
| SR-NF详细末级 → SysReq-NFR架构末级 | 每条 SR-NF 详细末级分配到唯一 SysReq-NFR 架构末级 | ✅ 通过 |
| SysReq场景活动 → PA末级 | 每条 SysReq 9级活动分配到唯一 PA 末级 | ✅ 通过 |

### 违规记录

无违规。所有层间映射均满足 1:1 分配约束。
