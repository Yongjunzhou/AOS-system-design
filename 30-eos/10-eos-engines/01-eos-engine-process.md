# 流程引擎

**状态**：待开发

## 职责

提供业务流程的定义、执行和监控能力，支持审批流、工作流等多种流程模式。

## 提供的核心能力

| 能力 | 说明 |
|------|------|
| 流程设计器 | 可视化定义流程节点、转换条件和路由规则 |
| 流程执行引擎 | 流程实例的创建、推进、挂起、终止 |
| 节点类型 | 审批节点、任务节点、条件分支、并行网关等 |
| 流程监控 | 流程实例跟踪、超时告警、效能分析 |
| 流程版本管理 | 流程定义的版本控制和灰度发布 |

## 配置产出类型

- 流程定义（BPMN）
- 流程模板
