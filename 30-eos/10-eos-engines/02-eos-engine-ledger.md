# 台账引擎

**状态**：待开发

## 职责

提供数据台账（列表/看板）的配置、展示和操作能力，是 EOS 中最核心的数据组织和展示方式。

## 提供的核心能力

| 能力 | 说明 |
|------|------|
| 台账设计器 | 配置列定义、筛选条件、排序规则 |
| 视图定义 | 列表视图、卡片视图、树形视图 |
| 数据聚合 | 汇总行、分组统计、交叉分析 |
| 行操作 | 行级按钮、批量操作、行内编辑 |
| 数据导出 | Excel/CSV 导出、打印 |

## 配置产出类型

- 台账定义（`20-eos-configs/01-eos-config-ledger.md`）
- 视图定义
