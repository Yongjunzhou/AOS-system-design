# 表单引擎

**状态**：待开发

## 职责

提供表单的配置、渲染、数据提交和验证能力。使用户可以通过配置而非编码的方式定义和管理数据录入/展示表单。

## 提供的核心能力

| 能力 | 说明 |
|------|------|
| 表单设计器 | 可视化拖拽配置表单布局和字段 |
| 字段类型体系 | 文本、数字、日期、选择、引用等 |
| 数据验证 | 字段级和表单级的校验规则 |
| 表单渲染 | 运行时渲染为 Web 界面 |
| 数据提交 | 表单数据的存储和更新 |

## 配置产出类型
