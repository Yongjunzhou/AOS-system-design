## 各 Skill 读写契约

### 通用操作规范：三段式循环

各技能对 22 号资产的操作统一遵循"使用→丰富→修订"三段式循环。单次技能执行内完成完整循环，无需跨技能传递。

```
技能执行开始
  │
  ① 使用资产 ────→ 加载 22 正文区，按需要筛选：
  │                  ├─ 生命周期状态=已确认 → 做匹配参考
  │                  ├─ 生命周期状态=候选/疑似 → 做比对/查重
  │                  └─ 生命周期状态=废弃 → 不做处理
  │                  └→ 对输入材料逐条比对
  │                      ├─ 匹配 → 标注 ID
  │                      └─ 不匹配 → 暂存"待丰富"
  │
  ② 丰富资产 ────→ 对暂存材料做分析/聚类/精炼
  │                  └→ 将可沉淀的新资产写入 22：
  │                      ├─ 新分类维度 → #### @cand/疑似/nfr-xxx
  │                      └─ 新具体需求 → ##### @xxx-REQ-xx
  │
  ③ 修订标注 ────→ 重新加载 22（刚写入的内容已可用）
  │                  └→ 用新资产标注之前未匹配的材料
  │
技能执行结束
```
