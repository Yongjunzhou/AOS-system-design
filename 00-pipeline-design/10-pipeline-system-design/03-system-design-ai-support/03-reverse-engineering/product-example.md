# 逆向工程系统设计：产品数据范例
**Reverse Engineering System Design: Product Data Example**

**用途**：展示一个完整的逆向工程系统设计过程的输入和输出范例，AI 参照此范例从代码和图纸中逆向推导设计文档。

**依据B说明**：本范例使用"运营体系开发之系统设计流水线"作为演示对象，展示逆向工程8步流程的具体产出。范例假设该流水线有完整的 Java/React 软件实现代码，需要从代码反向推导出五层设计文档。根据目标产品的类型（软件产品或方法论/工具产品），PA 分类可调整为相应的体系。

**范例项目**：运营体系开发之系统设计流水线（System Design Pipeline）

**场景设定**：系统设计流水线已开发完成，有完整的 Java Spring Boot 后端代码、React 前端代码和架构图纸，但五层系统设计文档缺失或不符合 v3.0 准则要求。现在需要从代码和图纸中逆向推导出完整的、符合准则的设计文档集。
