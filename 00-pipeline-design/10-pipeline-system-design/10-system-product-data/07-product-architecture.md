## 非功能约束

### 非功能需求对产品架构的约束

| SysReq-NFR | 约束 | 受影响的 PA 组件 |
|-----------|------|----------------|
| NFR-001 批量处理性能 | < 30分钟/1000条 | PA-001~010（核心引擎层）、PA-064~075（Skill 层） |
| NFR-002 单条处理性能 | < 1.8秒/条 | PA-001~010（核心引擎层） |
| NFR-003 系统可用性 | ≥ 99.9% | 全部在线组件 |
| NFR-004 故障恢复 | MTTR ≤ 15分钟 | PA-039（版本管理器）、全部在线组件 |
| NFR-005 传输加密 | TLS1.2+ | PA-082（API 网关）、全部网络通信 |
| NFR-006 存储加密 | AES-256 | 全部数据存储组件 |
| NFR-007 访问控制 | RBAC | PA-062（权限控制模块） |
| NFR-008 系统可靠性 | MTBF > 1000h | 全部核心组件 |
| NFR-009 故障率 | < 0.1% | 全部核心组件 |
| NFR-010 并发支持 | ≥ 100 并发 | PA-082（API 网关）、全部在线组件 |
| NFR-011 并发响应 | 95% ≤ 2秒 | PA-001~010、PA-064~075 |
| NFR-012 多语言翻译 | 覆盖率 100% | PA-084（文档站点）、全部前端组件 |
| NFR-013 语言切换 | 动态切换 | 全部前端组件 |
| NFR-014 单元测试 | ≥ 90% | 全部组件 |
| NFR-015 集成测试 | ≥ 80% | 全部组件间接口 |
| NFR-016 部署时间 | ≤ 30分钟 | CI/CD 管道 |
| NFR-017 部署频率 | ≥ 1次/天 | CI/CD 管道 |
