## 🎯 使用流程

### 第一次使用（30分钟）

1. **理解背景**（10分钟）
   - 阅读本README
   - 了解文档体系结构

2. **学习准则**（10分钟）
   - 阅读 `01-design-standards/01-system-design-standards.md` 的"核心概念"部分
   - 理解5层结构和3个核心约束

3. **选择指南**（10分钟）
   - 根据你的场景，选择合适的设计指南
   - 参考 `02-design-guidelines/README.md` 的场景对比表

### 深入学习（2-3小时）

1. **阅读完整准则**
   - 阅读 `01-design-standards/01-system-design-standards.md` 的全部内容
   - 理解每层的具体要求和质量指标

2. **阅读选定的指南**
   - 根据你的场景，阅读相应的设计指南
   - 理解操作流程和最佳实践

3. **参考示例**
   - 查看 `08-system-design-pipeline/` 中的实际例子
   - 理解如何应用准则和指南

### 日常使用

1. **快速查阅**
   - 使用 `02-design-guidelines/05-quick-reference-card.md` 快速查阅
   - 参考检查清单进行质量检查

2. **遵循流程**
   - 按照选定指南的步骤执行
   - 定期进行质量检查

3. **参考最佳实践**
   - 学习指南中的最佳实践
   - 应用到实际项目中
