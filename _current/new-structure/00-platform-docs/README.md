## 📖 各目录说明

### 01-getting-started/
**快速开始**

为新用户提供快速入门指导。包含：
- 快速开始指南（5分钟快速了解）
- 最终总结（完整的文档集总结）

**推荐**：第一次接触系统设计流水线的用户

### 02-system-design-pipeline/
**系统设计流水线**

包含系统设计流水线的5层报告和相关分析文档。包含：
- 工作要求定义
- 原始需求及分析
- 相关方需求
- 系统需求
- 产品架构方案
- 需求映射矩阵

**推荐**：想要查看具体的设计文档的用户

### 03-design-standards/
**系统设计准则**

包含系统设计的理论基础和规范要求。包含：
- 系统设计准则（理论基础）
- 英文术语对照表（国际化支持）

**推荐**：想要深入理解方法论的用户

### 04-design-guidelines/
**系统设计实践指南**

包含系统设计的实践指导和操作步骤。包含：
- 三种场景的操作流程（详细指导）
- 快速参考卡（便携版）

**推荐**：想要学习如何进行系统设计的用户

### 05-documentation/
**文档体系和索引**

包含文档体系说明、索引和相关参考资料。包含：
- 文档体系说明
- 文档索引（快速查找）
- 文档总结
- 工作总结

**推荐**：想要了解文档体系的用户
