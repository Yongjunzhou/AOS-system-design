## 🎯 使用流程

### 第一次使用（1小时）
1. 阅读 `01-documentation-system.md`（15分钟）
2. 根据角色选择学习路径（5分钟）
3. 阅读相关的准则或指南（40分钟）

### 日常使用
1. 使用 `02-documentation-index.md` 快速查找
2. 根据需要查阅相关文档
3. 参考快速参考卡进行快速查阅

### 团队培训
1. 介绍文档体系（1小时）
2. 讲解系统设计准则（2小时）
3. 讲解系统设计实践指南（3小时）
4. 实际案例演练（4小时）
5. Q&A和最佳实践分享（2小时）
