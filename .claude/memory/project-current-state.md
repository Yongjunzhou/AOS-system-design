### 关键讨论结论

1. 架构末级节点 = 分类管理单元(容器)，不是详细条目
2. NFR架构节点 = 一个非功能需求主题(如"易用性"、"访问安全性")，按内容归类
3. SR-NFR保留角色维度(某类相关方的一个主题)，SysReq-NFR去掉角色(一个主题，跨角色归并)
4. 功能需求性能指标(如响应时间)≠纯NFR，不可混入NFR路径示例
5. 指标分解的三场景判定流(判断1/2/3)结构已建立
