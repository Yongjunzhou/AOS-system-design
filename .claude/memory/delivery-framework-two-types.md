# 交付物框架缩减为两类

**日期**：2026-06-02

## 变更内容

A1（元流水线）和 A2（EOS 流水线）规范全面从"四类构件"框架改为"两类交付物"框架。

| 旧框架（四类构件） | 新框架（两类交付物） |
|------------------|-------------------|
| 设计规范 | **设计规范** — 定义各产品的设计规范 |
| 设计指南 | ❌ 移除 |
| AI 技能文档 | **AI SKILL** — 可执行技能定义 |
| 任务定义 | ❌ 移除 |

## 涉及文件

- `10-pl4pleos/00-pl4pleos-spec/01-pl4pleos-spec-sysdev.md`（A1 规范）
- `20-pl4eos/00-pl4eos-spec/01-pl4eos-spec-sysdev.md`（A2 规范）

## 保留不变

- EOS 流水线（A2）的推导链锚点描述（A2 输出 EOS，EOS 的 PA 类型为软件组件，不受影响）
- A3（EOS）规范的交付物描述（非流水线类产品，框架不同）

A2：EOS 流水线系统设计子流水线的设计者和维护者
