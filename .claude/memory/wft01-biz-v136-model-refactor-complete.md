## wft01-biz v1.36 完成（2026-06-27）

### 模型变更

识别锚点从"主实体"转为"输出文档"（广义=业务产出物+指标产出物）。QTC 从属性升格为独立场景部件（指标产出物）。引擎判定拆为业务产出物三选一/指标产出物→指标引擎，删除"从属"概念。

### 架构图（方案A）

```
场景业务（L4）  ← 端到端业务闭环 + 场景级 QTC（聚合计算）
  ├── 场景部件 A  ← 业务产出物，@engine-表单引擎
  │   └── 场景部件 A-QTC1  ← 部件级指标产出物，@engine-指标引擎
  ├── 场景部件 B  ← 业务产出物，@engine-流程引擎
  │   └── 场景部件 B-QTC1  ← 部件级指标产出物，@engine-指标引擎
  └── 场景部件 E  ← 场景级指标产出物（聚合），@engine-指标引擎
```
