更新章节内的层级对照表后，必须逐节审查同章其余小节（含流程图、映射表、各步骤正文）中与表相关的术语和描述，确保全文一致。

**Why:** 本会话中重构 §7.1 层级表后，发现了 §7.4 的 `SysReq`→`SysReq-F` 不一致、§7.5 缺少"跨相关方归并"和"内嵌于5级节点"描述等多处问题——表变但文不变会导致内部矛盾。

**How to apply:** 层级表变更后，执行三段联动检查：① 同节内其余表格和图示 → ② 同章其他节的关键术语引用 → ③ 跨章交叉引用。
